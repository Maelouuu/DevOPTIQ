Cartographie — OptiqCarto À documenter

OptiqCarto est un outil de cartographie SVG développé en interne. Il existe en deux versions synchronisées : intégré dans DevOPTIQ (static/optiqcarto/) et standalone dans le repo OptiqCarto/.

À compléter : fonctionnement de l'éditeur, format JSON de sauvegarde, synchronisation DB, import VSDX, viewer lecture seule.
